# 宠物医院管理系统 java1221

## 项目概述

宠物医院管理系统是基于Spring Boot和Vue.js开发的，旨在为宠物医院提供全面的管理解决方案。本系统采用前后端不分离的模式，确保了系统的高效运行和便捷维护。

## 技术栈

- **后端：**
  - Spring Boot
  - MyBatis
  - JSP
  - MySQL 5.7

## 功能模块

### 系统角色

- 系统管理员
- 医生
- 用户

### 系统功能

- **系统管理：**
  - 系统设置
  - 页面管理
  - 角色管理
- **用户管理：**
  - 用户资料维护
- **宠物管理：**
  - 宠物列表
  - 宠物健康史
- **预约管理：**
  - 预约时间设置
  - 医生时间安排
- **日常健康：**
  - 健康指南
  - 健康监测
  - 健康标准
- **宠物档案管理**
- **医院管理：**
  - 医院信息维护

## 运行环境

- Java运行环境
- MySQL数据库

## 部署步骤

1. 安装Java运行环境。
2. 安装MySQL数据库，并导入初始数据。
3. 部署后端服务。
4. 部署前端页面。
